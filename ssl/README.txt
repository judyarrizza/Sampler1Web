The certificate.cnf, localhost.crt, and localhost.key files were created by following https://medium.com/@richardr39/using-angular-cli-to-serve-over-https-locally-70dab07417c8

You will have to add the localhost.crt to the Trusted Root Certificate Authorities so that it becomes a trusted certificate

To add the certificate to the Trusted Root Certificate Authorities you can follow the section of from the above link named "Trusting the certificate on Windows".
